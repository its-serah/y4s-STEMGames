Why You’ll Love It

A detailed and engaging game that teaches you about cybersecurity and ethical hacking.

Scientific Concepts Explored

  • A friendly explanation of advanced situations. It applies realistic cybersecurity knowledge and rules.
  • Realism is anchored in core cybersecurity principles like terminal use, scripting your own tools, and basic networking, without teaching real attack techniques. To keep it approachable, methods are gamified and simplified; as players learn the loop, it feels friendlier. [Interview, Oct 2025]
  • Players build terminal fluency and foundational networking, practicing real style commands (ls, cd, chmod, chown, mkdir, ssh, ftp). [Interview, Oct 2025]
  • Every tool is implemented in the game’s own scripting language, with readable/editable source. [Interview, Oct 2025]
  • This hands on tinkering reinforces core concepts and accelerates learning. [Interview, Oct 2025]

Learning Objectives

  • Learn: CyberSecurity / Ethical Hacking, plus terminal fluency and basic networking.
  • Practice: Cryptography, hacking, DDoS attacks, etc., including real style commands and reading/modifying command source.
  • Apply: Real challenges.
  • Encourage: Creativity / logical thinking.

Gameplay Overview

You start with a beginner challenge of connecting to WiFi by hacking your way through it. Later, you get advanced challenges like retrieving data by hacking into a computer using real life tools and commands, including missions like hacking a school system to change a client’s grade.

The founder acknowledges the steep learning curve, especially for newcomers to Linux or the command line. In version v1.0 of the game, a tutorial rework will further ease onboarding. [Interview, Oct 2025]

An early “aha” moment comes from running ScanLan, which instantly visualizes the local network, devices and connections. Seeing the topology clarifies how LANs are structured and how hosts relate. Beginners quickly grasp navigation and plan pivots between devices. [Interview, Oct 2025]

History & Fun Facts

  • Made by Loading Home, a single developer team led by Kurouzu from Spain.
  • Kurouzu envisioned a shared universe powered by a simulated operating system, inspired by a long standing interest in MMORPGs and cybersecurity. He kept the persistence and systems depth but removed nonessential Unix complexity to make it approachable. [Interview, Oct 2025]
  • This project was inspired by the classic hacking game Uplink, especially its “trace” mechanic from detection to attribution, as a core influence at earlier stages of development. The atmosphere and lore draw from movies like The Matrix and Mr. Robot. [Interview, Oct 2025]
  • Scripting is both the signature decision and the key reversal: what began as a non feature became indispensable through player feedback. Player scriptable tools now define the experience. [Interview, Oct 2025]
  • Fun quirk: an unexpected behavior in the whois command revealed admin details on player networks (mirroring NPC networks). The community found it compelling for sleuthing, leading the team to revert the fix and keep the dynamic in spirit. [Interview, Oct 2025]
  • Standout community creations include “Coins,” “Capture the Flag,” small games, bash replacements, and automation tools, many of which inspired integrated support. [Interview, Oct 2025]
  • Modding is limited in multiplayer to prevent cheating; single player supports small mods now, with plans to expand. [Interview, Oct 2025]
  • Community translations are available via Steam Workshop and are supported by the game. [Interview, Oct 2025]
  • Version 1.0 is near; most roadmap features are complete. Remaining items are music/audio, a tutorial rework, and bug fixing. [Interview, Oct 2025]
