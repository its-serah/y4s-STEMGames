Why You’ll Love It

If you enjoy solving puzzles and thinking your way through challenges, or if you love programming (or are just starting to learn it), then this game is for you.

Scientific Concepts Explored

  • The game introduces algorithms and control flow as you sequence actions with loops and conditionals.
  • You learn to manage state and decompose big goals into smaller functions for planting, watering, and harvesting.
  • You practice automation and optimization by replacing manual grind with efficient, resource aware scripts.
  • You develop debugging and iteration habits by changing code, observing drone behavior, and refining your approach.

Learning Objectives

  • Learn: The fundamental concept of automation.
  • Practice: Basics of Python and increasing code complexity as new gameplay elements are introduced.
  • Apply: Logical thinking to develop autonomous harvesting algorithms that adapt to expanding environments and obstacles.
  • Encourage: Problem solving and debugging skills by testing and refining player written scripts to achieve optimal efficiency.

Gameplay Overview

In The Farmer Was Replaced you use a simple Python based programming language to program an agricultural drone and harvest crops. As you progress, your landscape expands, new crops appear, and obstacles can block your way. Over time, the game gives you more advanced coding elements and lets you create increasingly complex code. Your objective is to automate harvesting as efficiently as possible.

Players see the drone execute their code immediately, fueling frequent “aha!” moments. Working programmers especially love the direct visualization. [Interview, Oct 2025]

How to Play

The game explains each element step by step and lets you build your own code. There is also an information button in the top right corner if you have questions about the game.

History & Fun Facts

  • Made by Timon Herzog, an independent developer from Switzerland with a computer science background, known for The Farmer Was Replaced and Assembly Planter.
  • The project began as a non Python like FarmDrone prototype and shifted toward a Python like language after players asked for a more authentic coding feel, while the title playfully suggests that your drone “replaces” the farmer by doing the work you code. [Interview, Oct 2025]
  • Early versions did not prioritize font/readability, color blind safety, or key remapping, which created additional work later. Localization has since become a priority to reach global audiences. [Interview, Oct 2025]
  • Unlike many programming games that focus purely on puzzles, this game combines simulation with an upgrade tree and incremental systems. [Interview, Oct 2025]
  • A Mac version is in development (timeline to be announced). [Interview, Oct 2025]

Easter Eggs

  • Beat the game to unlock a drone hat that’s basically the farmer’s skull. [Interview, Oct 2025]
